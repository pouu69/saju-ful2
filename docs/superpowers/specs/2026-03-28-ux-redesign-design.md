# UX 전면 리디자인 — 사주명리의 미궁

**날짜:** 2026-03-28
**목표:** B2C 배포 가능한 수준의 전체 UX 개선 + 사주 카드를 핵심 킥(Kick)으로 강화
**타겟:** 한국 + 일본 모바일 사용자 (20~40대)

---

## 1. 개요

### 현재 문제
- 랜딩 → 입력 → 결과의 단순 3단계 플로우로 임팩트 부족
- 사주 카드가 결과물의 일부로만 존재, 메인 훅(hook)이 아님
- 카드 공개 순간의 드라마가 약함
- 공유 흐름이 이미지 다운로드에만 의존, 바이럴 루프 없음
- 단일 폼에 모든 입력이 집중되어 몰입감 없음

### 목표
- 카드를 서비스의 핵심 약속으로 랜딩에서부터 노출
- 의식(儀式)처럼 단계적으로 쌓이는 기대감
- "SNS에 올리고 싶다", "소장하고 싶다", "친구에게 보내고 싶다"는 반응 유도
- DB 없이 공유 링크 기능 구현

---

## 2. 새 UX 플로우

```
히어로 랜딩
  ↓ "카드 받기" CTA 탭
대화형 입력 (6단계)
  ↓ 마지막 단계 완료
운명 계산 로딩
  ↓ 계산 완료
봉투 수령 (새 단계)
  ↓ 봉투 탭하여 개봉
카드 공개 + AI 해석
  ↓ "공유 링크 복사" 탭
공유 페이지 (/share/[token])
  ↓ 친구가 링크 열기
"나도 카드 받기" → 바이럴 루프
```

---

## 3. 랜딩 페이지 (B2: 큰 카드 + 특징 설명)

### 레이아웃 (모바일 퍼스트)

**상단 히어로 배너 (배경: radial-gradient gold glow)**
- 브랜딩: "AI · 四柱命理" (소형 레터스페이스)
- 카드 미리보기 (중앙, 88px 폭):
  - 오행 4기둥 컬러 그리드 (木=초록, 火=빨강, 水=파랑, 金=금색)
  - 띠 이모지 + 간지 표기
  - AI 요약 한 줄 (이탤릭)
- 타이틀: "나의 사주 카드"
- 서브: "나만의 운명 카드를 받아보세요"

**하단 패널 (background: #0d0b00)**
- 특징 3줄 리스트 (아이콘 + 제목 + 설명):
  - 🔮 AI 사주 해석 — GPT 기반 개인 맞춤 풀이
  - 🃏 사주 카드 증정 — 저장·공유 가능한 나만의 카드
  - ⚡ 1분 완성 — 생년월일시만 입력하면 끝
- CTA 버튼: `linear-gradient(135deg, #D4A020, #FFD060)`, "지금 무료로 받기"

### ASCII 인트로 처리
기존 ASCII 인트로 애니메이션은 **CTA 버튼 탭 후 로딩 전 단계**에 짧게 삽입하여 보존.
랜딩 자체에서는 표시하지 않음.

---

## 4. 대화형 입력 플로우 (6단계)

각 단계는 전체 화면을 차지하며, 상단에 **6칸 진행 바** 표시.
단계 전환 시 슬라이드 또는 페이드 트랜지션 적용.

| 단계 | 질문 | UI 컴포넌트 |
|------|------|------------|
| ① | 성함이 어떻게 되십니까? | 터미널 스타일 텍스트 입력 (금색 언더라인) |
| ② | 태어난 해는 언제인가요? | 드럼 롤러 (선택 시 간지·띠 즉시 표시) |
| ③ | 태어난 월과 일은? | 숫자 입력 또는 드럼 롤러 |
| ④ | 양력인가요, 음력인가요? | 큰 2택 버튼 |
| ⑤ | 태어난 시간을 아시나요? | 시진 그리드 (🌙子時~🌆戌時 + ❓모름) |
| ⑥ | 성별과 결혼 여부는? | 토글 버튼 쌍 |

**뒤로가기**: 각 단계 상단 좌측에 ← 버튼으로 이전 단계 복귀 가능.

## 5. 운명 계산 로딩

**중앙 ☯ 글로우 서클** (radial-gradient, pulsing animation)
텍스트: "운명을 읽는 중..."
서브: "사주팔자를 계산하고 AI가 해석하고 있습니다"

**단계별 진행 표시 (체크리스트 스타일)**:
1. ✓ 사주팔자 계산 완료 (green)
2. ◐ 오행 균형 분석 중... (gold, pulsing)
3. ○ 카드 생성 대기중 (dim)

**기술 구현**: 사주 계산(`calculateFullSaju`)은 즉시 완료되므로,
AI API 호출(`streamInterpretation`)과 병렬로 진행하며
로딩 화면은 최소 1.5초 표시 (UX 기대감 유지 목적).

---

## 6. 봉투 수령 (신규 단계)

현재 플로우에 없는 **새 중간 단계**. 카드 공개 전 마지막 빌드업.

- 화면 중앙: 봉인된 봉투 컴포넌트
  - 봉투 배경: `linear-gradient(160deg, #2a1e08, #1a1200)`, gold border
  - 봉인 씰: 원형 레드 배경 + ☯ 기호 + gold ring
- 텍스트: "{이름}님의 운명의 카드가 도착했습니다"
- 안내: "봉인을 여세요 / 탭하면 열립니다"
- 탭 시 → 봉투 열리는 애니메이션 → 카드 공개 페이지로 전환

## 7. 카드 공개 (강화)

### 카드 비주얼 개선

기존 카드에 **오행 4기둥 컬러 그리드** 추가:

```
[甲/木 초록] [丙/火 빨강] [壬/水 파랑] [庚/金 금색]
   년주         월주         일주         시주
```

각 셀: 천간(대) + 오행(소) + 배경 컬러 (해당 오행 색상 15% opacity)

카드 글로우 강화:
```css
box-shadow:
  0 0 40px rgba(212,160,32,0.5),
  0 0 80px rgba(212,160,32,0.2),
  0 12px 32px rgba(0,0,0,0.7);
```

### 공개 시퀀스 (기존 유지 + 봉투 추가)
1. GoldParticles gather → burst (기존)
2. 봉투 개봉 애니메이션 (신규)
3. 카드 낙하 (card-draw-down, 기존)
4. 3D 플립 (rotateY 180deg, 기존)
5. 글로우 강화 후 정착

### 공개 후 액션 버튼
```
[💾 저장]  [🔗 공유 링크 복사]
```
공유 링크 복사 시 toast: "링크가 복사되었습니다!"

## 8. 공유 링크 구조 (DB 없음)

### URL 형식
```
/share/[token]
token = base64url(생년월일+시간+성별+역법)  // 이름 제외, ~20자
```

**인코딩 데이터 (최소화)**:
- 생년(4) + 생월(2) + 생일(2) + 시간인덱스(2) + 성별(1) + 역법(1) = ~12바이트
- base64url 인코딩 → ~16~20자

**제외 데이터**: 이름 (프라이버시), 전체 AI 텍스트 (URL 길이)

### 공유 페이지 (`/share/[token]/page.tsx`)

1. URL에서 token 추출 → base64url 디코딩
2. 사주 재계산 (`calculateFullSaju`) — 이름 없이 계산
3. **AI 호출 없음** — 기존 fallback template(`src/lib/ai/templates.ts`) 사용
4. 이름 대신 "庚午年生 말띠의 사주" 형식으로 표시
5. 항목별 요약 카드로 구성 (전체 풀이 아님, 아래 참조)
6. 하단: "나도 카드 받기 ✦" → `/` 로 이동 (바이럴 CTA)

### 공유 페이지 표시 항목 (요약 형태)

전체 AI 해석 대신, fallback template에서 항목별 핵심 1~2줄만 추출하여 카드 형태로 표시:

| 섹션 | 표시 내용 | 출처 |
|------|---------|------|
| 🃏 사주 카드 | 오행 4기둥 컬러 그리드 + 띠 | `calculateFullSaju()` |
| 🌿 오행 성향 | 주 오행 키워드 + 한 줄 설명 | `ELEMENT_PERSONALITY[dominant].keyword` |
| ☯ 일간 성격 | 자연물 비유 + 핵심 테마 한 줄 | `DAY_MASTER_METAPHOR[dayMaster].theme` |
| 🍀 오행 균형 | 강한 오행 / 부족한 오행 표시 | `fiveElements` 분포 |
| 🔒 전체 풀이 잠금 | "AI 해석은 직접 받아야 볼 수 있어요" | CTA로 유도 |

**핵심 UX 원칙**: 공유 페이지는 "티저"로 설계. 충분히 흥미롭되, 깊은 해석은 직접 받아야만 볼 수 있어 "나도 받기" 클릭을 자연스럽게 유도.

**OG 태그** (SNS 공유 미리보기):
```html
<meta property="og:title" content="庚午年生 말띠 사주카드" />
<meta property="og:description" content="성장과 도전의 기운 — AI 사주명리의 미궁에서 나의 운명을 확인하세요" />
```
